## Why

项目当前缺少 README.md 文档，导致新贡献者和用户无法快速了解 Amphoreus 的定位、技术栈和运行方式。项目即将在 GitHub 上公开协作，需要一个中英双语 README 作为项目的入口文档。

## What Changes

- 新增 `README.md` 文件（项目根目录），包含中英双语内容
- 文档涵盖：项目简介、技术栈、功能特性、快速开始、项目结构、API 概览

## Capabilities

### New Capabilities

- `readme-documentation`: 为项目根目录创建中英双语 README.md，包含项目介绍、技术栈、快速开始指南和项目结构说明

## Impact

- 仅新增 `README.md` 文件，不影响任何现有代码
- 提升项目的可发现性和新成员上手体验
